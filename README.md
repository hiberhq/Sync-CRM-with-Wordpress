# Requirements
## Python 3
To check if it is installed:

`python3 --version`

## pip3
To check if it is installed:

`pip3`

Otherwise install them. The way depends on platform.

# Install dependecies
`$ pip3 install requests`

`$ pip3 install bs4`

`$ pip3 install python-dotenv`

# Usage

`$ python3 p.py`


## Hint
Run script in background:

`nohup python3 p.py &`

Check python processes:

`ps ax | grep p.py`
